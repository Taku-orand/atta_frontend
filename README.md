# 目指した課題解決
このアプリケーションは特に以下のような悩みを持つ人を想定しています。
- よく落とし物をする
- 落とし物を探してるけど見つからない

このアプリケーションでは、アイテムを登録していただきQRコードを発行し落とし物にくっつけることで、落とし物をした際にもし誰かが見つけたときにQRコードから持ち主にお知らせを届けることができます。
